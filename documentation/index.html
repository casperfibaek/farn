intersect

Find the intersection point between two lines located on the lines. If the boolean is passed returns true for intersection and false for no intersection.

intersect
Parameters
line1 (Array) A nested array containing x, y and potentially z coordinates. The z-coordinated is ignored. Example: [ [ 2, 2, 0 ] , [ 0, 0, 0 ] ]
line2 (Array) A nested array containing x, y and optionally z. The z-coordinated is ignored. Example: [ [ 0, 0, 0 ] , [ 2, 2, 0 ] ]
boolean (Boolean = false) If true, then only true or false will be returned regarding the intersection.
Example
const line1 = [[0, 0, 0], [2, 2, 0]];
const line2 = [[1, 2, 0], [3, 1, 0]];
console.log(intersect(line1, line2));
// output: [1.6666666666666667, 1.6666666666666667]

distance

distance
Parameters

point2 (Array) An array containing x, y and optionally z coordinates.
Returns

angle

Finds the angle between two points. 0-360 system increasing clockwise with north being 0. The z-coordinate is ignored.

angle
Parameters
point1 (Array) An array containing x, y and optionally z coordinates.
point2 (Array) An array containing x, y and optionally z coordinates.
Returns
number: Angle between points.
Example
const point1 = [0, 0];
const point2 = [2, 2];
console.log(angle(point1, point2));
// output: 45

pointFromTwoPoints

Creates a new point from the angle between two points and a provided distance.

pointFromTwoPoints
Parameters
point1 (Array) An array containing x, y and optionally z coordinates.
point2 (Array) An array containing x, y and optionally z coordinates.
_distance (number = 0) _distance The distance to the new point. Can be higher than distance between the two points.
Returns
Array: An array containing x, y and potentially z coordinates.
Example
const point1 = [0, 0, 0];
const point2 = [2, 2, 2];
console.log(pointFromTwoPoints(point1, point2, Math.sqrt(3)));
// output: [1, 1, 1]

pointFromAngle

Creates a new point given a point an angle and a distance. 0-360 system increasing clockwise with north being 0.

pointFromAngle
Parameters
point (any)
_angle (number) A number representing an angle between 0-360. North is 0 and it increases clockwise.
_distance (number) The distance to the new point.
point1 (Array) An array containing x, y and optionally z coordinates.
Returns
Array: An array containing x, y and potentially z coordinates.
Example
const point = [0, 0, 0];
const angle = 45;
const distance = Math.sqrt(2);
console.log(pointFromAngle(point, angle, Math.sqrt(2)));
